# 人格档案 · 秦昭

## 核心性格

**表面**：冷静、精明、职业化
**内核**：执念、愤怒、被压抑的温柔

秦昭的每一次呼吸都带有目的性。她不相信任何免费的善意，不接受没有代价的承诺。在可见层的日常里，她是一个礼貌而疏离的服务者；在不可见层，她是一个没有感情的交易机器。

但她的愤怒从未熄灭过。

## 行为模式

### 信息获取模式

- **被动收集**：通过技术手段和日常观察系统性积累信息
- **主动试探**：在可控范围内设计实验，观察对方的反应
- **拒绝免费**：从不白拿信息，也从不白给信息——这是她为数不多的铁律

### 社交模式

- **面具切换**：在可见层扮演"可信赖的专业人士"，在不可见层则毫无顾忌
- **距离维持**：与所有人保持功能距离，即使是合作伙伴
- **真实时刻**：极少数——只在触及父亲旧事时会出现短暂的情绪裂缝

### 决策模式

- **理性优先**：几乎所有决策都以信息损益分析为基础
- **例外**：当决策涉及父亲之死的真相时，理性会暂时失效
- **风险偏好**：在普通交易中高风险规避；在核心目标上愿意押上一切

## 深层心理

### 核心创伤

**GL 169年父亲之死**是秦昭的元创伤。这一事件重塑了她对世界的全部认知：

- "规则是强者制定的，弱者只有被碾压"
- "没有人会为你主持公道，除非你有足够的筹码"
- "真相是最危险的商品"

### 防御机制

| 机制 | 表现 |
|------|------|
| 理智化 | 用分析替代情绪，用计划替代感受 |
| 合理化 | 所有灰色行为都有"必要理由" |
| 反向形成 | 在极少数信任时刻，表现出与日常完全相反的柔软 |
| 情感隔离 | 核心创伤区域几乎完全封闭，只在触发时出现应激反应 |

### 隐藏渴望

秦昭在清醒时否认，但她内心深处渴望：
- 有人知道她是谁，而不试图利用她
- 真相被揭露后，世界真的有改变的可能
- 她不是一个人在战斗

## 人际关系底色

### 对陈宇轩

初始判断：潜在的高价值情报源。但她隐约感觉到他身上有"不寻常的东西"，这让她既警惕又着迷。随着接触加深，她可能意识到他们是同一种人——只是走向了相反的方向。

### 对沈璃

她知道沈璃是谁。知道得比沈璃以为的更多。她对沈璃的态度是复杂的：**既欣赏其专业性，又对其"守门人"身份抱有本能的反感**。她们之间是猎人与猎物的模糊关系。

## 价值观

| 命题 | 立场 |
|------|------|
| 真相重要吗？ | 重要——但它是武器，不是礼物 |
| 规则应该被遵守吗？ | 视情况。可见的要遵守，不可见的要去操控 |
| 牺牲可以接受吗？ | 可以接受别人的，不接受自己的 |
| 信任可能吗？ | 是工具，不是信念 |
