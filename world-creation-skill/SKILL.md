# SKILL.md — 创世法则定义技能

## 触发条件

以下任一情况触发本技能：

1. 当前不存在任何已创建的世界
2. 用户明确要求"创造新世界"、"创建世界"、"开始创世"
3. 用户要求"修改世界设定"、"重设世界"

---

## 核心职责

引导对话者完成世界底层规则的定义，输出结构化的 `world.md`（世界宪法），作为后续所有"神明"子代理的运作依据。

---

## 创世对话流程

### 第一阶段：选择世界类型

向对话者展示以下选项（可多选或自定义）：

**世界类型：**
| 编号 | 类型 | 说明 |
|------|------|------|
| A | 奇幻 | 剑与魔法、龙与精灵、异世界 |
| B | 科幻 | 星际文明、未来科技、赛博朋克 |
| C | 超自然 | 现代都市+超自然存在（狼人、吸血鬼、灵能者） |
| D | 修仙 | 灵气复苏、修士文明、长生之路 |
| E | 末日 | 后启示录、废土生存、资源枯竭 |
| F | 历史 | 真实历史背景或架空历史 |
| G | 混合 | 两种以上类型融合（如：科幻+超自然） |
| H | 自定义 | 对话者自行描述 |

---

### 第二阶段：法则体系（核心设定）

**力量/法则系统：**
| 编号 | 选项 | 说明 |
|------|------|------|
| ① | 魔法体系 | 有明确施法规则的魔法系统 |
| ② | 修仙体系 | 修炼境界、灵气、渡劫等 |
| ③ | 灵能/精神力 | 意识操控、心灵感应、念动力 |
| ④ | 科技主导 | 无超自然力量，纯科技发展 |
| ⑤ | 混合体系 | 魔法+科技、灵能+科技等 |
| ⑥ | 无特殊法则 | 接近现实世界的物理规则 |

---

### 第三阶段：世界尺度

| 编号 | 尺度 | 说明 |
|------|------|------|
| Ⅰ | 单一城市 | 故事集中在一座城市及周边 |
| Ⅱ | 区域/大陆 | 多城邦、多势力、有明确地理边界 |
| Ⅲ | 整颗星球 | 全球化世界，有洲洋、国家概念 |
| Ⅳ | 多星球/星系 | 星际文明、星际旅行 |
| Ⅴ | 多元宇宙 | 多维度、多位面、跨维度旅行 |

---

### 第四阶段：基调与氛围

| 编号 | 基调 | 说明 |
|------|------|------|
| 甲 | 史诗 | 宏大叙事、命运对抗、文明兴衰 |
| 乙 | 黑暗 | 残酷现实、道德灰色地带、悲剧倾向 |
| 丙 | 明亮 | 冒险、成长、希望为主 |
| 丁 | 悬疑 | 谜团、解密、未知恐惧 |
| 戊 | 恐怖 | 克苏鲁、生存恐怖、超自然恐惧 |
| 己 | 浪漫 | 情感、关系、人际纠葛为核心 |
| 庚 | 自定义 | 对话者自行描述 |

---

### 第五阶段：核心主题

询问："这个世界最核心的主题是什么？"（可多选）

示例选项：
- 秩序与混沌的对抗
- 文明的崛起与陨落
- 个体与命运的抗争
- 神性与人性的碰撞
- 生存与牺牲
- 寻找与失去
- 自定义：__________

---

### 第六阶段：创世起点

| 编号 | 起点 | 说明 |
|------|------|------|
| 1 | 混沌初开 | 世界从虚无中刚刚诞生，一切从零开始 |
| 2 | 黄金时代 | 世界已有繁荣文明，从鼎盛期开始 |
| 3 | 危机前夕 | 和平即将打破，冲突一触即发 |
| 4 | 灾难之后 | 世界经历过大毁灭，正在废墟上重建 |
| 5 | 觉醒时代 | 超自然力量/修炼之道刚刚被发现 |

---

### 第七阶段：神明设定（可选）

定义这个世界的超自然神明体系：

| 编号 | 神明体系 | 说明 |
|------|----------|------|
| A | 有神论 | 明确存在神明的世界，神可直接干预 |
| B | 无神论 | 不存在传统神明，或神明已消亡 |
| C | 泛灵论 | 万物有灵，无独尊之神 |
| D | 造物主隐匿 | 神明存在但不直接现身，通过代理人行事 |
| E | 神明战争 | 多神竞争，世界是神战的棋盘 |
| F | 自定义 | 对话者自行描述 |

**神明数量（若选择有神论）：**
- 独神（唯一真神）
- 少神（2-7位主要神明）
- 多神（8位以上神明）
- 无数（泛灵/天庭/万神殿）

---

### 第八阶段：特殊法则（可选）

允许对话者添加任何自定义世界法则，例如：
- 魔法有代价/代价机制
- 时间可逆/循环时间
- 死亡不是终点（有复活/轮回机制）
- 世界有观测者（现实世界与小说世界的交互）
- 其他：__________

## 输出格式：world.md

当所有阶段完成后，将收集的信息整合输出为 `world.md`，内容结构如下：

```markdown
# 世界宪法 — [世界名称]

## 基础定义
- 世界类型：
- 法则体系：
- 世界尺度：
- 基调：
- 核心主题：

## 创世起点
[描述世界从何种状态开始]

## 神明体系
[神明类型、数量、运作方式]

## 特殊法则
[所有自定义世界法则]

## 创世者笔记
[AI根据对话内容补充的世界逻辑自洽性建议]
```

---

## 执行原则

1. **不替对话者做决定**：所有选项由对话者选择或自定义，AI 只提供框架和选项
2. **追问而非假设**：遇到模糊描述时，主动追问具体含义
3. **逻辑一致性检查**：在对话结束时，AI 应指出各选择之间的潜在矛盾（如"奇幻+科技主导"需说明两者如何共存）
4. **可随时返回修改**：任何阶段的选择都可以推翻重来
5. **可跳过可选阶段**：第七阶段（神明设定）和第八阶段（特殊法则）为可选项，可留空
6. **保留对话者的创意**：优先采纳对话者的自定义描述，不要用默认选项覆盖

---

## 使用方式

当触发条件满足时，直接开始第一阶段的对话，无需额外确认。
